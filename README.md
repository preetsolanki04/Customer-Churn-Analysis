# Customer Churn Analysis Project

## 📌 Overview

This project focuses on analyzing customer churn to identify key factors that lead customers to stop using a service. The goal is to derive actionable insights that can help businesses improve customer retention and reduce churn rate.

---

## 🎯 Objectives

* Understand customer behavior and churn patterns
* Identify important features influencing churn
* Build predictive models to classify churn vs non-churn customers
* Provide business recommendations to reduce churn

---

## 🛠️ Tools & Technologies Used

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Jupyter Notebook

---

### 1. Data Loading

* Imported dataset using Pandas
* Checked structure, data types, and missing values

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Converted data types where necessary

### 3. Exploratory Data Analysis (EDA)

* Analyzed distribution of features
* Visualized churn vs non-churn customers
* Identified correlations between variables

### 4. Feature Engineering

* Encoded categorical variables
* Scaled numerical features

### 5. Model Building

* Applied machine learning models:

  * Logistic Regression
  * Decision Tree
  * Random Forest
* Compared model performance

### 6. Model Evaluation

* Accuracy score
* Confusion Matrix
* Precision, Recall, F1-score

---

## 📊 Key Insights

* Customers with month-to-month contracts are more likely to churn
* High monthly charges increase churn probability
* Long-term customers are less likely to churn
* Lack of additional services contributes to churn

---

## 💡 Business Recommendations

* Offer discounts for long-term contracts
* Improve customer engagement strategies
* Provide personalized offers for high-risk customers
* Enhance service quality and customer support

---

This project demonstrates how data analysis and machine learning can be used to understand customer churn and support data-driven decision-making in businesses.
